🚀 Tickyiyo 
Este es un proyecto de desarrollo web frontend diseñado para la gestión integral de conciertos y eventos. Permite a los administradores gestionar el inventario de eventos y categorías, y a los clientes navegar, filtrar y realizar compras simuladas de tickets.

 Características Principales
 Módulo de Cliente
Cartelera Dinámica: Visualización de eventos con imagen, fecha, hora y precio.

Búsqueda Inteligente: Filtro en tiempo real por palabras clave en el nombre del evento.

Filtros Avanzados: Filtrado por Ciudad y Categoría (sincronizado con el administrador).

Detalle del Producto: Vista ampliada con descripción detallada del evento.

Carrito de Compras: * Gestión de cantidades (agregar múltiples tickets por concierto).

Cálculo automático de totales y subtotales.

Opción de cancelar/vaciar el carrito.

Formulario de Checkout: Captura de datos del cliente (ID, Nombre, Dirección, Teléfono, Email) con registro de fecha de compra.

 Módulo de Administrador
Gestión de Categorías: Crear y almacenar categorías que alimentan los filtros del cliente.

Gestión de Eventos: CRUD completo para añadir nuevos conciertos al sistema.

Persistencia de Datos: Uso de localStorage para mantener la información activa en el navegador.

 Tecnologías Utilizadas
HTML5: Estructura semántica para una mejor accesibilidad.

CSS3: Diseño responsivo con Flexbox, Grid y variables personalizadas.

Estética: Dark Mode con degradados inspirados en tecnología y diseño moderno.

JavaScript: Lógica de programación pura para la manipulación del DOM, filtros de arreglos (filter, find, reduce) y gestión de estados.

LocalStorage: Para la persistencia de datos local sin necesidad de un backend inicial.
